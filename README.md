# 在线计算器

一个简洁美观的在线计算器，支持基本运算和键盘输入。

---

## 🌐 在线访问

**https://glorykim0327.github.io/calculator/**

---

## 🎨 功能

| 功能 | 说明 |
|------|------|
| 基本运算 | 加 (+)、减 (-)、乘 (×)、除 (÷) |
| 取余 | 模运算 (%) |
| 小数 | 支持小数点输入 |
| 删除 | 删除最后一位 (DEL) |
| 清空 | 全部清空 (AC) |
| 键盘支持 | 可直接用键盘输入数字和运算符 |
| 响应式 | 手机/电脑都能用 |

---

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `0-9` | 输入数字 |
| `.` | 小数点 |
| `+ - * /` | 运算符 |
| `Enter` 或 `=` | 计算结果 |
| `Escape` | 清空 |
| `Backspace` | 删除最后一位 |

## 💡 使用提示

- 支持连续运算，计算完成后直接输入运算符可继续计算
- 除数为 0 时会显示 "错误"
- 自动处理浮点数精度问题
