# 在线计算器

## 🌐 公网访问方案

由于服务器网络限制，推荐以下 **3 种方案**：

---

### 方案 1：GitHub Pages（推荐 ⭐）

**步骤：**

1. 创建 GitHub 仓库（如 `calculator`）
2. 上传 `index.html` 文件
3. 进入 Settings → Pages → 选择 main 分支
4. 获得链接：`https://你的用户名.github.io/calculator/`

**优点：**
- ✅ 完全免费
- ✅ 永久有效
- ✅ 速度快
- ✅ 无需服务器

---

### 方案 2：Vercel / Netlify

**步骤：**

1. 访问 https://vercel.com 或 https://netlify.com
2. 注册/登录
3. 拖拽 `calculator` 文件夹到部署区域
4. 获得公网链接

**优点：**
- ✅ 一键部署
- ✅ 自动 HTTPS
- ✅ 免费额度够用

---

### 方案 3：本地访问（当前可用）

**内网链接：** http://172.17.55.2:8888/index.html

**适用场景：**
- 同一局域网内访问
- 测试和开发
